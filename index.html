Salmon Boilerplate

This boilerplate is Justine Win's custom version of Devtip's Starter Kit V2.

Salmon v2 uses HTML5 and CSS3 (Non-Gulp) and includes clean flex layout classes, optional bootstrap & skeleton grid, and what-not.

Grids

Choose whether you want to use bootstrap's or skeleton's grid then go to index.html file and comment out the grid's css file you will not use in the `< head >` tag

Flex layout

This page is using flex classes. I have included helpful comments on the helper.css file. Go ahead, experiment!

Getting started

Fork or clone, do whatever you want. If you're into Sass and Jade, you can use Salmon Boilerplate - a Jade, Sass, and Gulp version of this.
